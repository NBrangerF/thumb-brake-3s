# Hook Generator 首帧 UGC Prompt Playbook

来源：`analysis_outputs/image2image_prompt_playbook.md`、`analysis_outputs/marketing_clip_starter_30/seedance_video_prompt_playbook.md` 和 hook deep research 的可复用结论。

## 图生图参考策略

- 商品图是严格的产品身份参考，不是松散风格参考。
- 必须保留商品形态、包装比例、标签布局、管盖/瓶盖结构、主色块和可读品类信号。
- 多参考图需要分工；Hook Generator V1 只使用商品图作为 `product_identity`。

## UGC 首帧结构

- 输出是一张 9:16 短视频第一帧，不是海报和电商主图。
- 使用真实手机拍摄感：手持构图、自然室内光、轻微不完美、生活场景。
- 首帧只保留一个停滑机制、一个视觉冲突点、一个可回收的商品或痛点线索。
- 首帧必须有足够视觉冲突感：反差、未完成动作、近景证据、错位物件、尴尬反应或异常结果至少命中一种。
- 禁止渲染字幕、贴纸、水印、UI 标签和随机文字。

## 0-3 秒留存逻辑

- 0-1 秒：用表情、反差、未完成动作、近景证据或错位物件停滑。
- 1-2.5 秒：继续制造好奇、冲突、异常线索或痛点，不要进入完整广告解释。
- 相关钩子：2-3.5 秒让观众看懂商品与冲突的关系。
- 强刺激钩子：前半段优先 grab attention，商品可以只在结尾短暂出现，但刺激必须平台安全且可回收。
- Hook Generator 只生成 hook，不生成完整转化段。
